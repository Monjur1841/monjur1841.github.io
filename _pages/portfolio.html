📈 Financial Engineering & Data Science

Portfolio Optimization via PyPortfolioOpt

  • Core Toolkit: Python, PyPortfolioOpt, NumPy, Pandas, Matplotlib
  • Overview: Engineered an automated data pipeline to ingest multi-year, minute-level high-frequency stock data from the National Stock Exchange (NSE). Programmed constraints to calculate dynamic asset weights, plot efficient frontiers, and maximize Sharpe ratios. This repository serves as the empirical core for my peer-reviewed publication in the Canadian Journal of Business & Information Studies.

Business Analytics Architecture

Transformed raw financial values (ROA, ROE, Debt-to-Equity ratios) into interactive operational dashboards to evaluate financial health.

📊 Applied Econometrics

Gravity Model Panel Frameworks (Migration & PPP)

  • Core Toolkit: Stata, Panel Data Regressions, Diagnostic Scripting
  • Overview: Modeled international migration flows across South Asian countries by building a custom panel database in Stata. Coded and evaluated structural Gravity Equations while addressing underlying endogeneity vectors via robust empirical strategies.

Cobb-Douglas Production Estimation Pipeline

  • Core Toolkit: Stata, OLS Regression, Diagnostic Testing
  • Overview: Estimated macroeconomic production function parameters using log-log OLS frameworks. Scripted rigorous model health checks, specifically incorporating the Breusch-Pagan test for heteroskedasticity and multi-collinearity variance inflation diagnostics.
